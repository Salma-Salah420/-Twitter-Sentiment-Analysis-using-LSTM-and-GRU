# -Twitter-Sentiment-Analysis-using-LSTM-and-GRU
--------------------------------------------------


This project focuses on performing sentiment analysis on Twitter data using deep learning models, specifically LSTM (Long Short-Term Memory) and GRU (Gated Recurrent Unit) networks. The goal is to automatically classify tweets into sentiment categories such as positive, negative, or neutral based on their textual content.

The dataset undergoes thorough preprocessing, including text cleaning, tokenization, stopword removal, and sequence padding to prepare it for neural network training. Word embeddings are used to represent textual data in a numerical format suitable for deep learning models.

Two recurrent neural network architectures—LSTM and GRU—are implemented and trained to compare their performance in understanding contextual information in text sequences. Evaluation metrics such as accuracy, precision, recall, and F1-score are used to measure model effectiveness.

This project demonstrates how deep learning techniques can be applied to natural language processing (NLP) tasks, particularly sentiment analysis, and highlights the differences in performance between LSTM and GRU models when working with social media text data
